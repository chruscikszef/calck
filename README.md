# Kalkulator Kurwa

# English: 

An open-source calculator written in C

## Installation 
$ git clone https://github.com/chruscikszef/calck/calck.git
$ cd calck
$ make
$ make install

## Supported operating systems
macOS, GNU/Linux, FreeBSD, OpenBSD, NetBSD, Solaris, UNIX
